## Blocking 은 무엇이고, 언제 발생하는지?

**Blocking**

* 호출된 함수가 자신이 할 일을 마칠 때까지 제어권을 가지고서 호출한 함수에게 바로 돌려주지 않는 것.
* ex) 사용자가 입력할 때 nextInt() 메서드가 제어권을 가지고 있어 사용자가 입력을 완료해야만 다음 코드로 넘어감.
* 즉, 수행 결과가 끝날때까지 제어권을 가지고 있는 것.
